Centos7 安装 Nginx

Nginx简介

  • Nginx是一个高性能的HTTP和反向代理web服务器，具有占有内存少，并发能力强等优点

一. 安装所需环境

1. gcc安装

  • 安装 nginx 需要先将官网下载的源码进行编译，编译依赖 gcc 环境

2. PCRE pcre-devel 安装

  • nginx 的 http 模块使用 PCRE 来解析正则表达式

3. zlib 安装

  • nginx 使用 zlib 对 http 包的内容进行 gzip

4. OpenSSL 安装

  • nginx 支持 https 协议
